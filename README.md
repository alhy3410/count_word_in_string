# Count_word_in_string README


## Installation


Retrieve the included Gemfile and Run the following command
```
bundle install
```

## Usage

To use the app run
```
ruby app.rb
```
Navigate in your browser to localhost:4567. Once page loads enter the word in the first form that you'd like to check against the words in the second form. Once you have the word and the words to check against entered in the form, hit the send button. The next page should load with the number of times the first word is repeated.
